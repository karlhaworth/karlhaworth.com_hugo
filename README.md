## Resume Website for Karl Haworth using Hugo

### Local

#### Serve

```bash
make serve
```

#### Build

```bash
make build
```

#### Tailwind Updates

```bash
make tailwind
```

### CloudFlare Build

```bash
make build-cloudflare
```
